# README
## これはなに？
イベントを管理することができるアプリです。

大まかな機能は、以下になります。
- イベントの一覧、詳細ページ
- イベントの作成
- イベントの削除
- イベントの更新
- 絞り込み検索

## 使用言語、バージョン
- Rails バージョン7
- React バージョン18
- React Router バージョン6

### ビルド
```
yarn build
```

### 起動
```
bin/rails s
```
